Random Number Generator

This is an interface that generates up to 10 million numbers using a Mersenne Twister random number generator i made from scratch. The RNG was seeded with your date and time when you entered this webpage Currently I have it set to display the first 5 or less numbers generated, however it does generate all numbers requested and appends them to an array. There are many applications of an rng, but an example of a usecase is shown below.

Plinko (Click on it to play)

This is a live interface for a gambling simulation made using the random number generator above. Although this doesn't have physics like a typical plinko game, this means each branch has an equal liklihood, therefore the bin return rates at the bottom can be calculated using a binomial distribution, resulting in a perfect 100% return. This means as n balls becomes large, you should never lose any money! This is just a fun way to show the functionality of the RNG.
